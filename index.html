Weather Visualizations

The purpose of this analysis was to decipher how varying weather elements are affected as we move from cities at differing latitudes. How does the maximum temperature, humidity, cloudiness, and wind speed change as we move further away from the equator and as we move closer to the equator. The analysis was done using weather data from over 500 cities that we obtained from the Open Weather API.


In order to display the data analysis, we used Matplotlib to create scatterplots. The weather elements we took into consideration were as follows: a city's maximum temperature, humidity levels, cloudiness, and wind speeds. This site will show you the visualizations created from the analysis of each weather element and its relation to latitude. The site also has our dataset that we used, and the graphs can be used to see any correlations.
